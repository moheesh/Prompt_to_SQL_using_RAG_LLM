About This Project

The SQL Learning Assistant is an intelligent chatbot designed to help users learn and generate SQL queries from plain English questions. It combines multiple AI techniques to deliver accurate, optimized, and well-explained SQL statements.

At its core, the system uses Retrieval-Augmented Generation (RAG) with a knowledge base of 80,000+ question-SQL pairs, a fine-tuned TinyLlama model trained with LoRA for domain-specific SQL generation, and Google's Gemini API for query refinement and natural language explanations.

Whether you're a beginner learning SQL basics or an experienced developer looking to quickly prototype queries, this assistant bridges the gap between natural language and database queries with intelligent context awareness and step-by-step explanations.
